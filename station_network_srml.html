Solar Radiation Monitoring Laboratory (SRML) - University of Oregon¶

The Solar Radiation Monitoring Laboratory (SRML) at the University of Oregon has been providing solar radiation data for the Northeastern United States since 1975. The SRML monitoring station network consists of both high-quality stations that measure all three irradiance components at a 1-minute resolution, as well as stations with low quality instruments that only log measurements hourly. A full list of the 42 stations (including discontinued statins) can be found on the SRML website.

A list of the high-quality SRML stations can be retrieved from the station listing:

import pandas as pd

df = pd.read_csv('solarstations.csv', sep=';', encoding='latin1')
df = df[df['Network'] == 'SRML']

Data retrieval¶

Data from the SRML stations is stored in monthly files for each station and can be freely downloaded from their website. The data can also be downloaded programmatically using the pvlib-python library, specifically the read_srml_month_from_solardat function.

An example of how to download data from June 2020 from the Hermiston station is shown here:

import pvlib

df_he = pvlib.iotools.read_srml_month_from_solardat(
    station='HE',
    year=2020,
    month=6)

df_he.head()  # print the first 5 rows of data
